# Curso Back-End - Programa Bolsa Futuro Digital - # BFD RS 2 Bento Gonçalves 1

###  Título do Projeto:
Pokedex Batalha Pokemon

### Propósito:
Criar uma versão simplificada do jogo [Pokemon.](https://pt.wikipedia.org/wiki/Pok%C3%A9mon_%28s%C3%A9rie_de_jogos_eletr%C3%B4nicos)

### Descrição:
O usuário pode:
- Manter multiplos treinadores.
- Manter multiplos bolsos com até 6 pokemons cada.
- Executar batalhas entre 2 treinadores com até 6 pokemons cada.

### Funcionalidades:
- Integração com banco de dados.
- Consulta de API central.
- Edição de treinadores e bolsos.
- Batalhas entre pokemons.

### Tecnologias:
- nodejs
- express
-  jquery
- bootstrap5.3
- API externa => Dados serão fornecidos pela: [pokeapi](https://pokeapi.co) e [pokedex-promise-v2](https://github.com/PokeAPI/pokedex-promise-v2)
  
### Público-alvo
Treinadores pokemon em evolução.
